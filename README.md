## 📌 Descripción

**Orvantia** es una marca enfocada en el desarrollo de soluciones de software empresarial a medida. El objetivo principal es ayudar a empresas medianas y grandes a estructurar sus procesos, centralizar su información y escalar sus operaciones mediante tecnología.

El sitio web presenta los servicios principales de la marca, el proceso de trabajo, preguntas frecuentes y una sección de contacto conectada a Formspree.

## 📬 Formulario de contacto

El formulario del sitio está pensado para recibir solicitudes de clientes mediante **Formspree**.

Campos incluidos:

- Nombre completo
- Empresa
- Correo electrónico
- Teléfono
- Servicio de interés
- Mensaje

Para activar el formulario:

1. Crear una cuenta en [Formspree](https://formspree.io).
2. Crear un nuevo formulario.
3. Copiar el ID generado.
4. Reemplazar `YOUR_FORM_ID` dentro de `index.html`.

Ejemplo:

```html
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
```

## 🌐 Deploy con GitHub Pages

Este proyecto puede publicarse fácilmente usando **GitHub Pages**.

### Pasos generales

1. Subir los archivos al repositorio.
2. Entrar a la configuración del repositorio.
3. Ir a:

```txt
Settings → Pages
```

4. Seleccionar:

```txt
Deploy from a branch
```

5. Elegir la rama:

```txt
main
```

6. Seleccionar la carpeta raíz:

```txt
/root
```

Una vez configurado, el sitio estará disponible en:

```txt
https://orvantia.github.io
```
