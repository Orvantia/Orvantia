## 📌 Descripción

**Orvantia** es una marca enfocada en el desarrollo de soluciones de software empresarial a medida. El objetivo principal es ayudar a empresas medianas y grandes a estructurar sus procesos, centralizar su información y escalar sus operaciones mediante tecnología.

El sitio web presenta los servicios principales de la marca, el proceso de trabajo, preguntas frecuentes y una sección de contacto conectada a Formspree.

---

## 🚀 Sitio web

Este repositorio contiene el código fuente del sitio web oficial de Orvantia.

El sitio está diseñado como una landing page profesional orientada a conversión, con secciones claras para presentar servicios, explicar el proceso de trabajo y facilitar el contacto con posibles clientes.

## ⚙️ Proceso de trabajo

El enfoque de Orvantia se basa en un proceso claro de principio a fin:

1. **Diagnóstico**  
   Análisis de la empresa, procesos actuales y objetivos del proyecto.

2. **Diseño**  
   Definición de la solución ideal de acuerdo con necesidades, alcance y presupuesto.

3. **Desarrollo**  
   Construcción iterativa con entregas parciales para validar avances.

4. **Lanzamiento**  
   Implementación asistida y capacitación para el equipo.

5. **Soporte continuo**  
   Mantenimiento, mejoras y evolución del sistema.
